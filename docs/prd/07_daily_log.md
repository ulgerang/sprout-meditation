# PRD: Daily Meditation Log Details

## 1. 개요 (Overview)
- **화면명**: Daily Meditation Log Details (일일 명상 상세 기록)
- **목적**: 과거의 특정 날짜에 수행한 명상 세션들과 작성한 기록을 타임라인 형태로 한눈에 확인 가능하도록 함.
- **주요 가치**: 성찰의 피드백, 과거 기록 검색.

## 2. 주요 기능 (Key Features)
### 2.1. 날짜 탐색
- **표시**: 현재 보고 있는 날짜를 크게 표시 (예: October 24, 2023).
- **달력 진입**: 우측 상단 달력 아이콘을 클릭하여 날짜 이동 가능.

### 2.2. 세션 타임라인
- **시간별 기록**: 오전, 점심, 저녁 등 시간 순서대로 명상 이력을 카드 형태로 나열.
- **정보 요약**: 시작 시간, 총 수행 시간, 세션 이름 표시.

### 2.3. 노트 열람 및 수정
- **노트 요약**: 작성된 일기 내용을 미리보기로 제공.
- **감정 태그**: 세션 종료 시 체크했던 감정(예: Calm, Focused) 표시.
- **관리 메뉴**: 각 세션 우측의 'more' 버튼을 통해 삭제나 공유 기능 제공.

### 2.4. 일일 통계 합계 (Daily Total)
- **총합 바**: 화면 하단에 오늘 수행한 명상 시간의 총합을 고정하여 표시.

## 3. UI 컴포넌트 및 레이아웃 (Components & Layout)
- **Layout**: 세로형 타임라인 레이아웃 (좌측에 수직 선과 포인트 점 위치).
- **컴포넌트 리스트**:
    - `TimelineCard`: 배경 아이콘, 제목, 기록 내용, 태그가 포함된 정보 카드.
    - `PrimaryTag`: 감정을 나타내는 아이콘과 함께 표시되는 칩(Chip) 스타일 컴포넌트.
    - `DailySummaryBar`: 하단에 플로팅 처리된 총계 요약 바.
    - `AddButton`: 새로운 기록을 수동으로 추가할 수 있는 버튼.

## 4. 기술적 고려사항 (Technical Details)
- **레이아웃**: `line-clamp-2`를 사용하여 긴 노트 내용은 말줄임표 처리.
- **인터랙션**: 전체 페이지 스크롤 시 상단 헤더와 하단 요약 바는 브라우저 블러 처리가 된 상태로 고정.
- **테마**: 시스템 다크/라이트 모드 자동 전환 대응.
