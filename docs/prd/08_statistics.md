# PRD: Meditation Statistics

## 1. 개요 (Overview)
- **화면명**: Meditation Statistics (명상 통계)
- **목적**: 사용자의 명상 습관을 데이터로 시각화하여 보여줌으로써 장기적인 목표 달성을 돕고 자기 효능감을 높임.
- **주요 가치**: 성장의 가시화, 동기 부여, 패턴 분석.

## 2. 주요 기능 (Key Features)
### 2.1. 월간 수행 현황 (Heatmap Calendar)
- **달력 표시**: 한 달간의 날짜를 그리드 형태로 표시.
- **활동 표시**: 명상을 수행한 날짜 하단에 점(dot) 또는 강조색으로 활동 여부를 표시.
- **탐색**: 화살표 버튼으로 이전/다음 달 이동.

### 2.2. 주간 요약 차트 (Weekly Summary)
- **막대 그래프**: 월~일요일까지의 명상 시간을 막대 그래프로 시각화.
- **성장 지표**: 지난주 대비 명상 시간 변화율(예: +12%) 표시.

### 2.3. 최근 세션 기록 (Recent Sessions)
- **목록**: 최근 수행한 세션 3~5개를 리스트 형태로 표시.
- **상세**: 각 항목 클릭 시 해당 날짜의 일일 로그 상세로 이동.

## 3. UI 컴포넌트 및 레이아웃 (Components & Layout)
- **Layout**: 위에서 아래로 스크롤하며 정보를 소비하는 리포트 구조.
- **컴포넌트 리스트**:
    - `CalendarWidget`: 깔끔한 디자인의 월간 활동 체크 달력.
    - `WeeklyBarChart`: 주간 데이터를 시각화한 막대 그래프 컨테이너.
    - `StatCard`: 최근 세션의 아이콘, 제목, 시간 범위, 총 시간이 포함된 카드.
    - `StickyBottomNav`: Stats 메뉴가 활성화된 하단 네비게이션 바.

## 4. 기술적 고려사항 (Technical Details)
- **데이터 바인딩**: 차트 막대의 높이는 실제 데이터 비율(`h-[N%]`)에 따라 동적으로 조절되어야 함.
- **다크 모드**: 차트의 보조색상 및 배경 블러 처리가 어두운 환경에서도 조화롭게 표시되도록 조정.
- **애니메이션**: 그래프 로드 시 막대가 아래에서 위로 올라오는 부드러운 트랜지션 적용 권장.
